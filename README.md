## Deskripsi
Website ini merupakan simulasi layanan pengiriman paket menggunakan drone (DroneXpress).  
Pengguna dapat mengisi form pemesanan, memilih kategori layanan (regular/express), serta tipe member (silver/platinum) yang akan mempengaruhi total pembayaran.

## Fitur
- Input data user (username, email, password)
- Pilihan jumlah paket
- Kategori layanan
- Sistem diskon berdasarkan member
- Perhitungan total pembayaran otomatis
